## Aula 2

Operadores lógicos e estruturas de decisões (IF e ELSE).

### Estrutura:
~~~python
if condição1 :
  print("Atende a condição 1");
elif condição2:
  print("Atende a condição 2");
else:
  print("Não atende nenhuma das duas condições");
  

~~~
